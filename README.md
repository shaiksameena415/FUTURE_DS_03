# FUTURE_DS_03 – Marketing Funnel Analysis

## Project Overview
This project analyzes marketing and lead funnel data to understand how users progress from visitors → leads → customers.

Objective:
- Identify funnel drop-off points
- Analyze high-performing marketing channels
- Improve lead-to-customer conversion
- Recommend actions to optimize the marketing funnel

---

## Dataset
Dataset used: Funnel / Marketing Campaign Dataset  
Includes columns like:
- Visitor channel (`contact`)
- Campaign month (`month`)
- Conversion status (`y`)
- Other engagement metrics

---

## Tools Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib

---

## Steps Performed

### 1. Data Loading
Loaded the dataset using Pandas.

### 2. Data Cleaning
- Checked for missing or inconsistent values
- Standardized categorical variables
- Formatted dates and month columns

### 3. Exploratory Data Analysis
Analyzed:
- Overall conversion rate
- Conversion by marketing channel
- Funnel drop-off points
- Monthly conversion trends

---

## Key Insights
- Overall conversion rate: **11.27%**
- Conversion by channel:
  - Cellular: ~14.7%
  - Telephone: ~5.2%
- Simplified funnel shows most visitors do not convert into customers.
- Conversion fluctuates month-to-month; certain months have higher success rates.

---

## Business Recommendations
- Invest more in the **cellular channel** for marketing campaigns.
- Optimize middle stages of the funnel to reduce drop-offs.
- Implement nurturing campaigns for leads that stall mid-funnel.
- Track monthly trends to plan campaigns during high-conversion months.
